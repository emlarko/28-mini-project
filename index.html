Flexbox Row

Use these three properties to create a Flexbox row layout.

.row {
                    display: flex;
                    flex-direction: row;
                    flex-wrap: wrap;
                    }

Flexbox Column

.column{
                        display: flex;
                        flex-direction: column
                    }
